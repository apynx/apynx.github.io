Ini adalah posting pertamaku di Apynx.github.io. Situs ini dibuat dengan bahasa pemrograman Jekyll. Terima kasih untuk Github yang telah menyediakan space hosting yang membuat situs ini dapat mengudara.

Saya pribadi sih sebenarnya baru tahu beberapa hari ini kalau Github ternyata memiliki fitur 'Pages'. Biasanya sih cuma pakai untuk menyimpan script-script website.

Dan ternyata, cara pakai Github Pages ini mudah sekali. Kamu hanya perlu mempelajari bahasan pemrograman sederhana seperti HTML, CSS, serta beberapa perintah Jekyll dan markdown.

Cukup klik tombol "Fork" dan Voilla,... desain template gratis dari orang lain langsung berada di repositori-mu. Lumayan, dapat gratisan subdomain dari Github juga lho..

Oh, iya. Theme yang dipakai di situs ini adalah '[Carte Noire](https://github.com/jacobtomlinson/carte-noire)' buatan [Jacob Tomlinson](http://carte-noire.jacobtomlinson.co.uk/).

Template ini awalnya berwarna gelap (dark theme) namun saya ubah menjadi terang dengan mengubah beberapa script CSS. Ada juga beberapa baris script yang saya hapus dan ubah agar tampilannya bisa lebih baik dan responsif.

Sayang template yang cantik ini tak lagi dilanjutkan pengembangannya sejak tahun 2015. Tapi semoga dengan desain yang cantik dan minimalis ini dapat menggugah semangatmu untuk ikutan membuat Github Pages. Salam!
